# React init pack

[![CircleCI](https://circleci.com/gh/C4co/react-init-pack.svg?style=svg)](https://circleci.com/gh/C4co/react-init-pack)

Initial configuration for npm react packages.

### Install
```
  $ yarn install react-utl
```

### Development

formating with prettier
```
  $ yarn format
```

tests
```
  $ yarn test
```

build
```
  $ yarn build
```
